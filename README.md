# Rock_paper_scissor:
The famous game of rock, paper and scissor
# Input:
Here, at first both the players will input their names.
Then, they will input a series of digits.
Then, they will ascertain the position of the digit which is to be considered.
Turn by turn the numbers would be written and hence, no chance of any cheating.
# Conditions:
The number 0, 1 and 2 are designated for rock, paper and scissor but not respectively rather in random predetermined order which is different for different players.
